## iNaturalist [![Build Status](https://github.com/inaturalist/inaturalist/workflows/inaturalist%20CI/badge.svg)](https://github.com/inaturalist/inaturalist/actions)

Open source Rails app behind [iNaturalist.org](https://www.inaturalist.org/)

Want to help out? Fork the project and check out the [Contributing Code to iNaturalist](https://github.com/inaturalist/inaturalist/blob/main/CONTRIBUTING.md). If you hit problems getting set up, post in our [Github discussions](https://github.com/inaturalist/inaturalist/discussions). This will ping our engineering team and someone will get back to you. We're a small team, so response might be slow.

Thinking about running your own version of iNaturalist? Consider joining the [iNaturalist Network](https://www.inaturalist.org/sites/network) instead of forking the community.

## Bugs and feature requests

**Please don't open GitHub issues for bug reports or feature requests.**

We track our work internally and triage inbound requests through the [iNaturalist Forum](https://forum.inaturalist.org). Compared to an issue here, you're more likely to get a fast response — and to hear from other community members with similar experiences — on the Forum.

* Bug reports: [forum.inaturalist.org Bug Reports category](https://forum.inaturalist.org/c/bug-reports/13)
* Feature requests: [forum.inaturalist.org Feature Requests category](https://forum.inaturalist.org/c/feature-requests/16)

For **security vulnerabilities** that require confidential communication, please email help+security@inaturalist.org directly. We don't offer bug bounties, but we do appreciate responsible disclosure.
